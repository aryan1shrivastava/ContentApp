# Content Calendar App - Spring Boot 3 Application

A practical Spring Boot 3 project built while learning backend development.

## 📌 Project Overview
This is a **Content Calendar Application** that helps manage content (blogs, videos, posts, etc.) with different statuses and publication dates.

Built as a hands-on learning project to understand Spring Boot structure and best practices.

## 🛠️ Tech Stack
- **Java 21**
- **Spring Boot 3**
- **Spring Data JDBC**
- **H2 Database** (in-memory)
- **Maven**

## 🎯 What I Learned

- Spring Boot project structure and auto-configuration
- Layered Architecture (Controller → Service → Repository)
- Using **Spring Data JDBC** (instead of JPA)
- REST API development with controllers
- Basic CRUD operations
- Exception handling in Spring Boot
- Application properties and configuration

## 🚀 How to Run

```bash
# Clone the repository
git clone https://github.com/aryan1shrivastava/ContentApp.git

# Navigate to project folder
cd ContentApp

# Run the application
./mvnw spring-boot:run
